# CACHE 数据源

![CACHE](../../../../img/new_ui/dev/datasource/cache.png)

- 数据源：选择 CACHE
- 数据源名称：输入数据源的名称
- 描述：输入数据源的描述
- IP 主机名：输入连接 CACHE 的 IP(如有多个IP，请用`,`分隔)
- 端口：输入连接 CACHE 的端口
- 用户名：设置连接 CACHE 的用户名
- 密码：设置连接 CACHE 的密码
- 数据库名：输入连接 CACHE 的数据库名称
- Jdbc 连接参数：用于 CACHE 连接的参数设置，以 JSON 形式填写

## 是否原生支持

是，数据源不需要任务附加操作即可使用。

> InterSystems Caché® 是一款高性能数据库，
